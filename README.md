# Rywils Tap

## How do I install these formulae?

`brew install rywils/tap/<formula>`

Or `brew tap rywils/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "rywils/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
